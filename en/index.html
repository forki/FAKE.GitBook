FAKE.GitBook

Fake.GitBook is a FAKE extension for gitbook using FSharp.Formatting.

The project is hosted on GitHub where you can report issues, fork the project and submit pull requests. The library is available under the MIT License, For more information see the License file in the GitHub repository.
